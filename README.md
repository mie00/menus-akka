# akka-menus

A basic rest api to query restaurants

## Requirements

Scala v2.12.4
Sbt v0.13.16

## Installation

```bash
git clone https://github.com/mie00/menus-akka.git
cd menus-akka
```

update the configuration file [src/main/resources/application.json](./src/main/resources/application.json).

```bash
sbt run
```
